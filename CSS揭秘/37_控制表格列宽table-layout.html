css2.1中，table-layout默认为auto,是自动表格布局算法，还可以接受fixed，是固定表格布局算法。

在使用时，只需要对table元素或具有display:table的元素使用table-layout属性即可，为了确保属性奏效，需要为表格元素指定一个宽度(那怕是100%)。

同样，为了让text-overflow:ellipsis奏效，我们也需要为那一列指定宽度。
