### 弹窗
1. SimpleDialog
    ```dart
    SimpleDialog(
      title: Text('请选择你的性别'),
      titlePadding: EdgeInsets.only(),
      contentPadding: EdgeInsets.only(),
      children: <Widget>[
        SimpleDialogOption(
          child: Text('男'),
          onPressed: () {
            Navigator.pop(context);
          },
        ),
      ],
    )
    ```
1. AlertDialog
    ```dart
    showDialog(
      context: context,
      builder: (_) => AlertDialog(
        title: Text('提示'),
        content: Text('这是一个Dialog！'),
        actions: <Widget>[
          FlatButton(
              onPressed: () {
                Navigator.of(context).pop();
              },
              child: Text('取消')),
          FlatButton(
              onPressed: () {
                Navigator.of(context).pop();
              },
              child: Text('确定'))
        ],
      )
    )
    ```
